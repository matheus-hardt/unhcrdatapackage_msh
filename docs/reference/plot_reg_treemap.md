# Plot Regional Treemap

Plot Regional Treemap

## Usage

``` r
plot_reg_treemap(
  year = 2024,
  region = "The Americas",
  pop_type = c("REF", "ASY", "IDP", "OIP", "OOC", "STA"),
  label_font_size = 15
)
```

## Examples

``` r
plot_reg_treemap(
    year = 2024,
    region = "The Americas",
    label_font_size = 15
)
```
