# Plot Regional RSD

Plot Regional RSD

## Usage

``` r
plot_reg_rsd(
  year = 2024,
  region,
  top_n_countries = 10,
  measure = "Recognized",
  category_font_size = 10
)
```

## Examples

``` r
plot_reg_rsd(

    region = "The Americas",
    measure = "Recognized",
    category_font_size = 10
)
```
