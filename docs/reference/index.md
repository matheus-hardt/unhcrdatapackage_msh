# Package index

## Chart for Country

Charts adressing questions at country level

- [`plot_ctr_asylum()`](plot_ctr_asylum.md) : Asylum Applications &
  Decision over time This charts allow to visualize trends in terms of
  asylum applications, decision and refugee status recognition
- [`plot_ctr_destination()`](plot_ctr_destination.md) : Main Destination
  from one specific countr
- [`plot_ctr_diff_in_pop_groups()`](plot_ctr_diff_in_pop_groups.md) :
  Increases and Decreases in Population Groups
- [`plot_ctr_disp_migrant()`](plot_ctr_disp_migrant.md) : Plotting ratio
  Refugee/Asylum-seekers vs Migrant within a country
- [`plot_ctr_keyfig()`](plot_ctr_keyfig.md) : Summary Key Figures Key
  figures can be highlighted with humanitarian icons
  https://fontawesome.com/icons/categories/humanitarian
- [`plot_ctr_location()`](plot_ctr_location.md) : Map of Destination
  Countries
- [`plot_ctr_origin_history()`](plot_ctr_origin_history.md) : Plot
  Origin History
- [`plot_ctr_origin_recognition()`](plot_ctr_origin_recognition.md) :
  Display A Chart with Refugee Recognition Rate for nationals from a
  specific country
- [`plot_ctr_population_type_abs()`](plot_ctr_population_type_abs.md) :
  Main country of origin - Absolute value
- [`plot_ctr_population_type_perc()`](plot_ctr_population_type_perc.md)
  : Main country of origin - Percentage
- [`plot_ctr_population_type_per_year()`](plot_ctr_population_type_per_year.md)
  : Graph of Population Type Over year
- [`plot_ctr_process()`](plot_ctr_process.md) : Asylum Processing
- [`plot_ctr_processing_time()`](plot_ctr_processing_time.md) : Plot
  Average processing time from registration to first instance asylum
  decision
- [`plot_ctr_pyramid()`](plot_ctr_pyramid.md) : Population Pyramid
- [`plot_ctr_recognition()`](plot_ctr_recognition.md) : Display a Chart
  with Refugee Recognition Rates
- [`plot_ctr_solution()`](plot_ctr_solution.md) : Plotting solutions
  within a country
- [`plot_ctr_solution_recognition()`](plot_ctr_solution_recognition.md)
  : Title
- [`plot_ctr_treemap()`](plot_ctr_treemap.md) : Tree map of Population
  Groups within a country
